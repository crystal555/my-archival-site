# About This Archive

This digital archive serves as a centralized repository for important documents, meeting records, and project materials. Built with Hugo and hosted on GitHub Pages, it provides a fast, searchable, and accessible way to preserve and share information.

## Features

- Static site generation for fast loading
- Mobile-responsive design
- Easy content management
- Version control through Git
- Free hosting on GitHub Pages
